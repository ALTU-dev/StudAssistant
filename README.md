# 🤖 Telegram AI Career Assistant Bot

A smart Telegram bot that helps users with career development, job search, resume creation, and interview preparation using AI technologies.

---

## 🚀 Overview

This project is a **Telegram bot** built in Python that integrates:
- AI-powered responses
- Job search functionality
- Resume generation
- User data collection

It is designed to guide users from **career exploration → job application → interview preparation**.

---

## ✨ Features

### 👤 User Registration
- Collects user information step-by-step
- Stores:
  - Name
  - Age
  - Skills
  - Experience
- Saves data in JSON format

---

### 📄 Resume Generator
- Automatically creates:
  - CV (Resume)
  - Cover Letter
- Based on user profile
- Ready for job applications

---

### 💼 Job Search
- Finds relevant job vacancies
- Filters:
  - Entry-level jobs
  - Internships
- Provides:
  - Job title
  - Company
  - Description

---

### 🎯 Career Recommendations
- Suggests suitable career paths
- Based on:
  - Skills
  - Interests
- Provides learning roadmap

---

### 🎤 Interview Preparation
- Generates interview questions
- Evaluates answers
- Gives feedback and improvement tips

---

## 🧠 AI Usage

The bot uses AI for:
- Text generation
- Career advice
- Resume writing
- Interview simulation

---

## 🛠 Technologies Used

- **Python**

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/telegram-career-bot.git
cd telegram-career-bot
```

### 2. Install dependencies
```bash
pip install pyTelegramBotAPI requests
```

### 3. Configure environment variables
Replace tokens in `main.py`:
```bash
BOT_TOKEN = "your_telegram_token"
API_KEY = "your_ai_api_key"
```

### ▶️ Run the bot
```bash
python main.py
```
